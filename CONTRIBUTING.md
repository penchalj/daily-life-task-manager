# 🤝 How to Contribute

We follow a standard professional workflow. Please follow these steps to contribute your code:

## 1. Claim an Issue
* Go to the **Issues** tab.
* Find an issue that isn't assigned to anyone yet.
* Comment `I would like to work on this` and wait for the teacher to assign it to you.

## 2. Create a Branch
Never work directly on the `main` branch. Create a new branch for your feature:
```bash
git checkout -b feature-your-task-name
