# 📝 Beginner Python: Daily Life Task Manager

Welcome to the **Task Manager** project! This is a collaborative Python application designed for students to learn the basics of Git, GitHub, and Python development.

The goal is to build a fully functional Command Line Interface (CLI) app that helps users manage their daily to-do lists using **CRUD** (Create, Read, Update, Delete) operations.

---

## 🚀 Getting Started

### 1. Prerequisites
Make sure you have **Python 3.x** installed on your machine. You can check by running:
`python --version`
